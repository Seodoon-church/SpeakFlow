# SpeakFlow 개발 로드맵

> 경쟁사 분석 기반 전략적 개발 계획
> 작성일: 2025-12-20

---

## 경쟁사 분석 요약

| 앱 | 강점 | 약점 | 가격 |
|---|---|---|---|
| 스픽 | AI회화, 실시간피드백 | 비쌈, 어휘약함 | 월1~2만 |
| 말해보카 | 게임화, 강제복습, 리그 | 회화없음 | 월8천 |
| 듀오링고 | 무료, 스트릭, 글로벌 | 회화약함, 초보답답 | 무료~1만 |

---

## SpeakFlow 포지셔닝

**스픽의 AI회화 + 말해보카의 게임화 + 저렴한 가격**

### 핵심 차별점
1. 가족 단위 학습 (멀티 프로필)
2. 트랙별 맞춤 (비즈니스, 여행, 학술, 키즈)
3. 한국인 특화 콘텐츠
4. AI 회화 + 단어 + 문법 통합

---

## Phase 1: 게이미피케이션 강화 (1주)

### 1.1 리그/랭킹 시스템
- [ ] 브론즈/실버/골드/플래티넘/다이아 리그
- [ ] 주간 리그전 (상위 10% 승급, 하위 10% 강등)
- [ ] 실시간 리더보드
- [ ] 리그별 보상 (젬, 배지)

### 1.2 스트릭 강화
- [ ] 스트릭 프리즈 (하루 패스)
- [ ] 스트릭 마일스톤 보상 (7일, 30일, 100일, 365일)
- [ ] 스트릭 복구 기능
- [ ] 친구와 스트릭 경쟁

### 1.3 일일 퀘스트
- [ ] 매일 3개 미션 (대화 1회, 단어 10개, 문법 1개)
- [ ] 퀘스트 완료 시 보너스 XP
- [ ] 주간 도전 과제

---

## Phase 2: 단어 학습 시스템 (2주)

### 2.1 단어장 구축
- [ ] 10,000+ 기본 단어 DB
- [ ] 레벨별 분류 (A1~C2)
- [ ] 카테고리별 분류 (일상, 비즈니스, 여행, 시험)
- [ ] 예문 + 발음 + 이미지

### 2.2 단어 퀴즈 (게임형)
- [ ] 4지선다 퀴즈
- [ ] 받아쓰기 퀴즈
- [ ] 듣고 맞추기
- [ ] 플래시카드 모드
- [ ] 타임어택 모드

### 2.3 SRS 복습 시스템 (말해보카 벤치마크)
- [ ] 망각 곡선 기반 자동 복습
- [ ] 오답노트 자동 생성
- [ ] 복습 알림 푸시
- [ ] 복습 완료율 추적

### 2.4 유니크 기능: AI 문맥 학습
- [ ] 단어를 대화에서 자연스럽게 사용
- [ ] AI가 배운 단어로 예문 생성
- [ ] 실제 회화에서 단어 활용 연습

---

## Phase 3: 문법 학습 시스템 (2주)

### 3.1 문법 코스 구축
- [ ] 초급/중급/고급 문법
- [ ] 한국인이 자주 틀리는 문법 특화
- [ ] 비디오 강의 (짧은 클립)
- [ ] 예문 + 연습문제

### 3.2 문법 퀴즈
- [ ] 빈칸 채우기
- [ ] 문장 순서 맞추기
- [ ] 오류 찾기
- [ ] 문장 만들기

### 3.3 유니크 기능: AI 문법 코칭
- [ ] 회화 중 문법 오류 실시간 교정
- [ ] 틀린 문법 자동 수집 → 맞춤 복습
- [ ] 문법별 약점 분석 리포트

---

## Phase 4: 콘텐츠 라이브러리 확장 (2주)

### 4.1 시나리오 라이브러리
- [ ] 50+ 롤플레이 시나리오
- [ ] 카테고리: 여행, 비즈니스, 일상, 면접, 데이트
- [ ] 난이도별 분류
- [ ] 사용자 시나리오 제작/공유

### 4.2 리스닝 콘텐츠
- [ ] 팟캐스트 스타일 오디오
- [ ] 받아쓰기 연습
- [ ] 속도 조절 (0.5x ~ 2x)
- [ ] 스크립트 + 번역

### 4.3 유니크 기능: 내 관심사 기반 학습
- [ ] 온보딩에서 관심사 선택 (영화, 음악, 스포츠, 요리 등)
- [ ] AI가 관심사 기반 대화 주제 생성
- [ ] 관심사별 단어장 자동 생성

---

## Phase 5: 유니크 학습법 (2주)

### 5.1 "오늘의 5분" 모드
- [ ] 바쁜 직장인을 위한 초단시간 학습
- [ ] 5분 안에 단어 5개 + 문장 3개 + 대화 1턴
- [ ] 지하철 모드 (음성 없이 학습)

### 5.2 "AI 튜터 매칭"
- [ ] 성격별 AI 튜터 (친절한 Emma, 엄격한 James 등)
- [ ] 튜터별 교수법 차이
- [ ] 튜터 레벨업 시스템

### 5.3 "실수에서 배우기"
- [ ] 내 실수 모아보기 대시보드
- [ ] 비슷한 실수 패턴 분석
- [ ] 실수 극복 미션

### 5.4 "가족 챌린지"
- [ ] 가족 구성원끼리 경쟁/협력
- [ ] 가족 합산 스트릭
- [ ] 가족 리그

---

## Phase 6: 수익화 (1주)

### 6.1 프리미엄 플랜
- [ ] 무료: 일일 3회 대화, 단어 20개

- [ ] 가격: 월 5,900원 (연 49,000원)

### 6.2 가족 플랜
- [ ] 4인 가족: 월 9,900원
- [ ] 프로필별 학습 관리
