# SpeakFlow v2.0 기획서

## 나를 닮은 AI와 함께하는 맞춤형 외국어 학습 플랫폼

**작성일:** 2025년 12월 18일
**버전:** 2.0 기획안

---

## 1. 프로젝트 비전

### 1.1 핵심 가치
> "내 얼굴을 한 AI 튜터와 함께, 나만의 속도로 외국어를 마스터하다"

### 1.2 차별화 포인트
| 기존 학습 앱 | SpeakFlow v2 |
|-------------|--------------|
| 일방적인 콘텐츠 제공 | 레벨 테스트 기반 맞춤 학습 |
| 텍스트/음성 챗봇 | **나를 닮은 AI 아바타 영상 대화** |
| 획일화된 커리큘럼 | 관심사별 시나리오 학습 |
| 단순 반복 학습 | 게이미피케이션 + 성취 시스템 |

---

## 2. 타겟 사용자

### 2.1 주요 타겟
- **일반 성인** (20~50대)
- 외국어 학습에 관심 있지만 시간이 부족한 직장인
- 여행/취미 목적의 가벼운 학습자
- 자격증/비즈니스 목적의 진지한 학습자

### 2.2 사용자 페르소나

| 페르소나 | 나이 | 목표 | 학습 시간 |
|----------|------|------|-----------|
| 여행러 민지 | 28세 | 일본 여행 회화 | 하루 10분 |
| 직장인 성훈 | 35세 | 비즈니스 영어 | 하루 30분 |
| 은퇴자 영숙 | 58세 | 중국어 취미 | 하루 20분 |

## 3. 핵심 기능

### 3.1 레벨 테스트 시스템

```
[신규 가입]
    ↓
[언어 선택] → 영어/일본어/중국어/스페인어...
    ↓
[레벨 테스트] (약 10분)
  ├── 어휘력 테스트 (객관식)
  ├── 문법 테스트 (빈칸 채우기)
  ├── 듣기 테스트 (음성 듣고 선택)
  └── 말하기 테스트 (음성 녹음 → AI 분석)
    ↓
[레벨 판정] → A1/A2/B1/B2/C1/C2 (CEFR 기준)
    ↓
[맞춤 커리큘럼 생성]
```

**레벨별 특징:**
| 레벨 | 설명 | 콘텐츠 |
|------|------|--------|
| A1 | 입문 | 기초 인사, 숫자, 자기소개 |
| A2 | 초급 | 일상 대화, 간단한 질문 |
| B1 | 중급 | 여행, 쇼핑, 의견 표현 |
| B2 | 중상급 | 뉴스, 토론, 비즈니스 |
| C1 | 고급 | 전문 분야, 관용표현 |
| C2 | 최상급 | 원어민 수준 대화 |

### 3.2 관심사별 맞춤 학습

**관심사 카테고리:**
- 여행/관광
- 비즈니스/직장
- 일상생활
- 문화/엔터테인먼트 (K-POP, 애니메이션, 드라마)
- 음식/요리
- 스포츠
- 기술/IT
- 의료/건강

**학습 콘텐츠 구성:**
```
[관심사: 여행]
├── 시나리오: 공항에서
│   ├── 체크인 대화
│   ├── 보안 검색대
│   └── 면세점 쇼핑
├── 시나리오: 호텔에서
│   ├── 체크인/체크아웃
│   ├── 룸서비스 요청
│   └── 컴플레인 하기
├── 시나리오: 레스토랑에서
│   ├── 예약하기
│   ├── 주문하기
│   └── 계산하기
└── 어휘집: 여행 필수 단어 500
```

### 3.3 AI 아바타 프리토킹 (핵심 차별화 기능)

**구현 방식:**
```
[사용자 프로필 사진 업로드]
    ↓
[AI 아바타 생성] (D-ID / HeyGen API)
    ↓
[아바타 커스터마이징]
  ├── 의상 선택
  ├── 배경 선택
  └── 음성 톤 선택
    ↓
[실시간 대화]
  ├── 사용자 음성 입력
  ├── Claude AI 응답 생성
  └── 아바타 영상 + 립싱크 출력
```

**아바타 기능:**
| 기능 | 설명 |
|------|------|
| 얼굴 복제 | 사용자 사진 기반 아바타 생성 |
| 립싱크 | AI 응답을 자연스러운 입모양으로 |
| 표정 | 감정에 따른 표정 변화 |
| 제스처 | 손동작, 고개 끄덕임 |
| 다국어 음성 | 원어민 발음 TTS |

**대화 모드:**
1. **프리토킹** - 자유로운 주제로 대화
2. **시나리오 롤플레이** - 상황극 (내가 손님, AI가 직원 등)
3. **발음 교정** - 내 발음 vs 원어민 비교
4. **인터뷰 연습** - 면접, 프레젠테이션 연습
5. **상황 시뮬레이션** - 사용자가 상상한 상황을 AI가 구현

### 3.5 상황 시뮬레이션 (신규 핵심 기능)

**사용자가 상황을 설명하면 AI가 즉시 시뮬레이션 생성!**

```
[사용자 입력 예시]
"도쿄 지하철에서 길을 잃었어요. 역무원에게 물어보고 싶어요"

    ↓ AI 분석

[시뮬레이션 자동 생성]
├── 배경: 도쿄 지하철 역
├── NPC: 일본인 역무원
├── 목표: 신주쿠역 가는 방법 묻기
├── 난이도: 사용자 레벨에 맞춤
└── 힌트: 필요한 표현/단어 제공
```

**시뮬레이션 예시:**

| 상황 입력 | 생성되는 시뮬레이션 |
|----------|-------------------|
| "파리에서 스테이크 주문하고 싶어" | 프랑스 레스토랑 + 웨이터 NPC |
| "미국 공항에서 짐을 잃어버렸어" | 공항 분실물 센터 + 직원 NPC |
| "일본 편의점에서 택배 보내고 싶어" | 편의점 + 알바생 NPC |
| "중국 병원에서 두통약 사고 싶어" | 약국 + 약사 NPC |
| "영어 면접에서 자기소개 해야해" | 면접실 + 면접관 NPC |
| "독일 맥주집에서 추천 받고 싶어" | 비어홀 + 바텐더 NPC |

**시뮬레이션 진행 방식:**
```
[시뮬레이션 시작]
    ↓
AI 아바타: "いらっしゃいませ。どちらへ行きたいですか?"
(번역 힌트: 어서오세요. 어디로 가고 싶으세요?)
    ↓
[사용자 음성/텍스트 응답]
    ↓
[AI 실시간 피드백]
├── 문법 체크 ✓
├── 발음 점수: 85점
├── 더 자연스러운 표현 제안
└── 다음 대화 이어가기
    ↓
[시뮬레이션 완료]
├── 종합 점수
├── 잘한 점 / 개선점
├── 이 상황에서 쓸 수 있는 추가 표현
└── 유사 시나리오 추천
```

**무한 확장성:**
- 정해진 시나리오가 아닌 **사용자 상상력 기반**
- AI가 상황에 맞는 NPC, 배경, 대화 흐름 자동 생성
- 같은 상황도 매번 다른 전개 가능
- 레벨에 따라 난이도 자동 조절

### 3.6 시나리오 라이브러리 (커뮤니티 공유)

**사용자가 만든 시나리오가 모두의 학습 자료가 된다!**

```
[시나리오 생성 → 공유 플로우]

사용자 A: "도쿄 라멘집에서 주문하기" 시나리오 생성
    ↓
[시뮬레이션 완료]
    ↓
"이 시나리오를 공유할까요?" → [공유하기]
    ↓
[시나리오 라이브러리에 등록]
├── 제목: 도쿄 라멘집에서 주문하기
├── 언어: 일본어
├── 레벨: A2~B1
├── 카테고리: 음식/레스토랑
├── 태그: #라멘 #도쿄 #주문
└── 제작자: 사용자A
    ↓
다른 사용자들이 검색/사용
```

**라이브러리 기능:**

| 기능 | 설명 |
|------|------|
| 검색 | 언어, 레벨, 카테고리, 태그로 검색 |
| 인기 순위 | 좋아요/플레이 수 기반 랭킹 |
| 즐겨찾기 | 마음에 드는 시나리오 저장 |
| 평점 & 리뷰 | 별점 + 후기 남기기 |
| 포크(Fork) | 다른 사람 시나리오를 내 버전으로 수정 |
| 크리에이터 보상 | 인기 시나리오 제작자에게 포인트/구독권 지급 |

**라이브러리 카테고리:**
```
시나리오 라이브러리
├── 🔥 인기 시나리오
├── ⭐ 에디터 추천
├── 🆕 최신 등록
├── 🎯 레벨별
│   ├── 입문 (A1)
│   ├── 초급 (A2)
│   ├── 중급 (B1-B2)
│   └── 고급 (C1-C2)
├── 🌍 언어별
│   ├── 영어
│   ├── 일본어
│   ├── 중국어
│   └── ...
└── 📂 카테고리별
    ├── 여행/관광
    ├── 비즈니스
    ├── 일상생활
    ├── 음식/레스토랑
    ├── 쇼핑
    ├── 의료/건강
    └── 긴급상황
```

**크리에이터 시스템:**
- 시나리오 100회 플레이 → 실버 크리에이터 배지
- 시나리오 1000회 플레이 → 골드 크리에이터 배지
- 월간 TOP 10 크리에이터 → Premium 1개월 무료
- 인기 시나리오 제작자 → 수익 쉐어 (향후 계획)

### 3.4 게이미피케이션 시스템

**경험치 & 레벨:**
```
[학습 활동]
├── 단어 학습: +10 XP
├── 문장 연습: +20 XP

[레벨업 보상]
```

**배지 시스템:**

**리더보드:**
- 주간/월간 랭킹

## 5. 기술 아키텍처

### 5.1 프론트엔드
```
React 19 + TypeScript
├── Zustand (상태관리)
├── Tailwind CSS (스타일)
├── Framer Motion (애니메이션)
├── React Router (라우팅)
└── PWA (모바일 앱)
```

### 5.2 백엔드
```
Supabase
├── Authentication (소셜 로그인)
├── Database (PostgreSQL)
├── Storage (이미지/음성 파일)
├── Edge Functions (서버리스)
└── Realtime (실시간 기능)
```

### 5.3 AI 서비스
```
AI Integration
├── Claude API (대화 생성, 문법 교정)
├── Whisper API (음성→텍스트)
├── ElevenLabs (텍스트→음성, 다국어)
├── D-ID / HeyGen (아바타 영상 생성)
└── OpenAI Vision (발음 분석용)
```

### 5.4 인프라
```
Deployment
├── Vercel (프론트엔드 호스팅)
├── Cloudflare (CDN, 보안)
└── AWS S3 (미디어 저장)
```
